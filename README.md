# DentalClinic Management System

## Описание

Система управления стоматологической клиникой для главврача, регистраторов, врачей и медсестёр.  
Позволяет управлять пациентами, записями на приём, врачами, услугами, документами и статистикой.

## Функционал

- Управление пациентами (поиск, добавление, редактирование)
- Расписание приёма (перенос пациентов между статусами: записан, ожидает, на приёме, завершён)
- Управление персоналом (врачи, регистраторы, медсестры)
- Управление услугами и ценами
- Создание документов и счётов
- Статистика по врачам, пациентам и услугам
- Календарь расписания
- Drag & Drop для перемещения пациентов

## Технологии

- Python 3.13.7
- Django 6.0.1
- Bootstrap 5
- JavaScript (AJAX, Drag & Drop)
- SQLite (или PostgreSQL)

## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone <ссылка_на_репозиторий>
   cd dental-clinic

2. Создайте виртуальное окружени
   python -m venv venv
   source venv/bin/activate # Linux/Mac
   venv\Scripts\activate # Windows

3. Установите зависимости:
   pip install -r requirements.txt

4. Запустите миграции
   python manage.py migrate

5. Загрузите начальные данные (если есть)
   python manage.py loaddata initial_data.json

6. Создайте суперпользователя:
   python manage.py createsuperuser

7. Запустите сервер:
   python manage.py runserver

Настройка
Откройте http://127.0.0.1:8000/admin/ и войдите как суперпользователь
Добавьте информацию о клинике в разделе ClinicInfo
Создайте врачей, регистраторов, услуги
Используйте http://127.0.0.1:8000/dashboard/ как главную страницу для главврача
Иконки и логотипы
Логотип: app/static/images/logo.jpg
Фавиконка: app/static/images/favicon.ico
Если файлы отсутствуют, замените на свои с теми же названиями.
Администрирование
Панель администратора: http://127.0.0.1:8000/admin/
Главврач: http://127.0.0.1:8000/dashboard/
Врач: http://127.0.0.1:8000/doctor/
Регистратор: http://127.0.0.1:8000/reception/
Медсестра: http://127.0.0.1:8000/nurse/
